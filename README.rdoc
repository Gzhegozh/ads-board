Rails version 4.2.4
Ruby version 2.3.0
PostgreSQL version 9.3.10

Run rails on port 3030:
rails s -p 3030

For the Angular frontend, I'm using a Yeoman generator and Gulp.
To install It run:

npm install -g yo bower gulp generator-gulp-angular

From app directory run:

npm install
bower install

Then run:

gulp serve
